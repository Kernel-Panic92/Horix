💡 Recomendaciones
📐
Dimensiones ideales
Ancho máximo de 180px, altura de 40–50px. El sistema escala automáticamente.
🎨
Fondo transparente
Usa PNG con fondo transparente para que se vea bien tanto en tema oscuro como claro.
💾
Almacenamiento
El logo se guarda en el servidor y se aplica a todos los usuarios automáticamente.
↩️
Restaurar original
Si eliminas el logo, el sistema vuelve a mostrar el texto HORIX
